Creating Anchor tags: < a >

Anchor tag is used to crate links. The text we want to display on the document page will have to written in between this anchor tags.
To create a link to this anchor we need to write the address of the link in the href attribute.
If we click on the anchor tag link it will load the address in the href atttribute.
